# Fake News Detection (Word2Vec + Support Vector Classification)

## Deskripsi
Proyek deteksi berita hoaks berbasis Word2Vec dan SVM dengan pipeline AWS.

## Arsitektur
Diagram

## Pipeline AWS
- Collecting → Python + Pandas
- Storage → AWS S3
- Processing → EC2 / SageMaker
- Accessing → Matplotlib

## Cara Run
pip install -r requirements.txt
